Por dentro do HTML

Antes de escrever uma única linha de código, todo desenvolvedor precisa entender o que o HTML realmente é — e não apenas decorar tags. Esta pesquisa vai te guiar por textos confiáveis sobre o assunto e pedir que você explique, com suas próprias palavras, o que aprendeu.

15 questões 8 fontes de leitura leitura + prática + discursiva
<section id="instrucoes">

01 Como funciona esta pesquisa

passo 1

Abra os links da seção "Material de pesquisa". Cada um corresponde a uma ou mais perguntas — está indicado ao lado de cada questão.

passo 2

Leia com atenção. Você não precisa entender 100% do texto técnico — o objetivo é captar a ideia central e conseguir explicá-la.

passo 3

Responda com suas próprias palavras. Cópia direta do site não conta como resposta — o que importa é que você entendeu.

passo 4

Nas questões práticas, escreva o código HTML pedido. Pode ser à mão, no caderno, ou testado em um editor — o que importar é o código estar correto.

03 Questões

01
leituradiscursiva
fonte: L1

O que significa a sigla HTML e qual é a sua função na construção de uma página web?

02
discursiva
fonte: L1

O texto explica que HTML não é uma linguagem de programação, e sim uma linguagem de marcação. Qual é, então, a diferença entre as duas? Por que essa distinção importa?

03
leitura
fonte: L2

O que é um "elemento" HTML? Explique as partes que normalmente o compõem (tag de abertura, conteúdo e tag de fechamento), usando um exemplo do material.

04
prática
fonte: L2

Escreva a estrutura básica completa de um documento HTML5: doctype, <html>, <head> com <title>, e <body>. Pode escolher qualquer assunto para o título.

05
leituradiscursiva
fonte: L2

O material explica o "aninhamento" de elementos: quando uma tag está dentro de outra, é preciso fechar primeiro a que foi aberta por último. Explique com suas palavras por que <p><strong>texto</p></strong> está incorreto.

06
leituradiscursiva
fonte: L2

Segundo o material, qual é a diferença entre elementos de bloco e elementos em linha (inline)? Cite um exemplo de cada um mencionado no texto.

07
prática
fonte: L3

Usando os elementos de título explicados no material, crie uma estrutura com um título principal e dois subtítulos sobre um tema de sua escolha (um hobby, um jogo, uma matéria, etc.).

08
prática
fonte: L3

Crie um parágrafo com no mínimo três linhas e, dentro dele, destaque uma palavra em negrito e outra em itálico, usando as tags corretas.

09
leituradiscursiva
fonte: L4

O material apresenta elementos semânticos para estruturar diferentes áreas de um site (cabeçalho, navegação, conteúdo principal, etc.). Por que usar esses elementos específicos é melhor do que usar apenas <div> para tudo?

10
leitura
fonte: L4

Cite três elementos estruturais mencionados no texto que costumam aparecer em quase todo site, e diga para que serve cada um.

11
leituradiscursiva
fonte: L5

O material explica a diferença entre link interno e link externo. O que caracteriza cada um? Por que os links internos são importantes para a navegação de um site?

12
prática
fonte: L6

Crie um link para o site do MDN (https://developer.mozilla.org) que abra em uma nova aba e tenha um atributo title explicando para onde o link leva.

13
prática
fonte: L7

Insira uma imagem (pode usar um caminho fictício, como foto.jpg) com um atributo alt que descreva corretamente o conteúdo da imagem.

14
leituradiscursiva
fontes: L7 e L8

Por que é importante preencher o atributo alt de uma imagem? O que acontece quando ele é deixado vazio de propósito (alt="")?

15
discursiva
reflexão final

Depois de pesquisar nos materiais indicados, explique com suas próprias palavras: por que aprender HTML é útil mesmo para quem não pretende ser programador?
